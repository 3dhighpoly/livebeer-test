# LiveBeer Test App

Тестовое приложение для программы лояльности LiveBeer.

## Стек
- Kotlin
- Jetpack Compose
- Navigation Compose
- Material3

## Экраны
- Welcome screen
- Phone input
- Verification code
- Registration

## Функциональность
- ввод номера телефона
- отправка кода подтверждения
- таймер повторной отправки
- регистрация пользователя

## Архитектура
Single Activity + Navigation Compose.

UI построен на Jetpack Compose.
